# Team-Krypton-OJT-SEM-2-Project-Portfolio
Team Agency Portfolio for Team Krypton


The central hub for our Semester 2 Advanced JavaScript Capstone, introducing our team and showcasing our suite of 5 functional applications built entirely without frameworks.

Features & Tech Stack:

Built with HTML5, CSS3, and Vanilla JavaScript, this portfolio demonstrates:

Dynamic DOM Rendering from JS data structures.

State Management (Light/Dark mode) using localStorage.

Form Validation for client-side email formatting checks.

Responsive Design adapting from mobile to desktop via CSS Flexbox.

Dynamic AI Mockups via URL prompting.

The Core Projects Suite:

Interactive Quiz App

Expense Tracker

Live News Feed

GitHub Explorer

Kanban Task Board
